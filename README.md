# DarkDev
A dark vscode theme for the people who like it dark.

## Installation
 1. Open Extensions on the sidebar
 2. Search for "**darkdev**"
 3. Install it
 4. Reload the editor
 5. Press ⌘K + ⌘T
 6. Pick "**darkdev**" one from the list and hit enter
 7. Enjoy!
